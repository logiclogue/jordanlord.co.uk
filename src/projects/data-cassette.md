Read and write files to a cassette tape!

## Installation

`npm install -g data-cassette`

Requires `aplay`.

## Write a file

Press record on your tape recorder.

`data-cassette -w test.jpg`

## Load a file

`data-cassette -r > test.jpg`

Press play on your tape recorder.
